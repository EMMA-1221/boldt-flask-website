Understanding the Success Estimate

The project team is only a component of what brings project success. Because of this, the estimates in the table are moderately accurate.

Any prediction with a CIM greater than or less than 10% is likely an outlier, which will occur in about 10% of the predictions.

Because of this, it is suggested to only look at the 'Result' of the table, and to try and change team members until you get an 'in 3rd quartile of teams' or 'in top 1/4 of teams' result.

From testing, the tool correctly guesses an above average team 64% of the time. 18% of the time it predicts a well performing team to be a below average team, and 18% of the time it predicts a below average team to be an above average team.

In short, by aiming for a higher performing team, the hope is the average team success will rise.

Team Building Hints

The tool is based on linear models that take an input from all the entered information and Drake P3 data and attempts to predict the project success.

If you're having trouble 'moving the needle' and finding a good team, checkout the suggestions below. Unfortunately, some categories didn't seem to have many specific attributes that effected success and it may be hard to find a better performing team.

The full model relies on elements from the Project Manager and Project Coordinator - try changing the staffing to this role.
The commerical model relies only on the number of employees per million dollars. Can increase number of employees staffed to project.
The industrial model relies only on the number of employees per million dollars. Can increase number of employees staffed to project.
The healthcare model relies on a number of variables and should change easily by adjusting the proposed team.
The power model relies only on the number of employees per million dollars. Can increase number of employees staffed to project.
The institutional model relies on elements from the Project Manager - try changing the staffing to this role.
The $1M-3M model relies on elements from the Project Manager - try changing the staffing to this role.
The $3M-10M model relies on a number of variables and should change easily by adjusting the proposed team.
The $10M+ model relies on eleements from the Project Manager, Project Executive, and the team as a whole. Try decreasing the project manager and project executive's dominance scores.
The less than one year model only relies on the distance from the corporate office and cannot be changed.
The greater than one year model relies on elements from the Project Manager, PRoject Engineer, Project Coordinator, and Project Executive. Try decreasing the project managers conscientiousness, increasing the project engineers primary extroversion, increasing the project coordinators environmental extroversion, decreasing the project coordinators environmental patience, and increasing the project executives environmental patience.
The lump sum model relies only on the number of employees per million dollars. You can try increasing the number of employees staffed to project.
The Time and Material model relies on a number of variables and should change easily by adjusting the proposed team.
The Guarenteed Max Price model relies on the Superintendent and Project Coordinator. Try adjusting these roles; specifically decreasing superintendent environental dominance.
The Time and Material with a GMP model relies on the Project Manager, Project Engineer, Superintendent, Project Executive, and the overall distance of the team from their homes. Try adjusting these roles to find a better team.
